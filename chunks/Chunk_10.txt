10. Chunk
Chunk title: EPA side-by-side comparison workflow

Source title: FuelEconomy.gov — Compare Cars Side-by-Side

Source URL: https://fueleconomy.gov/feg/Find.do?action=sbsSelect

 

Summary

FuelEconomy.gov offers a side-by-side comparison tool that allows up to four vehicles to be compared. For Cars.ai, this chunk supports a response pattern where the assistant organizes candidate vehicles into a comparison table or structured summary. It is especially useful for questions like “Which of these three cars is better for the long term?” because it encourages direct comparison rather than isolated descriptions. This chunk should be paired with NHTSA and IIHS safety chunks so that Cars.ai can compare efficiency and safety in the same answer. It also helps keep answers consistent: when more than one vehicle is under consideration, the assistant should compare the same evidence categories for each one instead of giving uneven or opinion-heavy descriptions.
